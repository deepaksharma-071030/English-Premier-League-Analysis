English Premier League Analysis Dashboard | Power BI

Developed an interactive English Premier League Analysis Dashboard using Power BI to analyze team performance, standings, and match statistics. The dashboard provides insights into points table rankings, wins, draws, losses, goal difference, and overall season performance.

Key Features:

📌 Dynamic Points Table Visualization with ranking logic
🎯 Highlighted zones for Champions League, Europa League, and Relegation positions
📈 Team performance comparison using interactive charts
🔍 Slicers for filtering by team and season
📊 Data cleaning and transformation using Power Query

This project demonstrates strong skills in data modeling, DAX calculations, data visualization, and dashboard storytelling.
